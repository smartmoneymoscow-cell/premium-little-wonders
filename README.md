# Premium Little Wonders

Разработка B2B/B2C интернет‑магазина детских товаров (личный кабинет, админка, корзина, оплата, ИИ‑консультант)

---

## Описание

Требуется разработчик для создания современного B2B/B2C интернет‑магазина по продаже детских товаров. Проект включает: адаптивный фронтенд на современном стеке, надежную админ‑панель, личные кабинеты для покупателей и оптовиков, корзину и оформление заказа, интеграцию платёжных шлюзов и ИИ‑консультанта для помощи покупателям. Каталог товаров — более 5000 SKU с фильтрами, атрибутами и импортом/синхронизацией прайс‑листов.

---

## Ключевые требования

### Стек
- React/Next.js или похожий современный SPA/SSR, TypeScript, Tailwind CSS (предпочтительно).

### Бэкенд
- Node.js/Express, NestJS или аналог; REST/GraphQL API; база данных — PostgreSQL или эквивалент.

### Админка
- Управление товарами, категориями, запасами, заказами, клиентами и ролями.

### Личный кабинет покупателей
- История заказов, оплат, адреса, личные скидки/оптовые условия.

### B2B
- Отдельные цены/каталоги для оптовых клиентов, прайс‑листы, минимальные партии.

### Каталог
- Поддержка 5000+ товаров, импорт CSV/XML, массовые правки, быстрый фильтр и поиск.

### Оплата и доставка
- Интеграция с платёжными шлюзами (например, Stripe/Яндекс.Касса/Тинькофф), калькуляция доставки, статусы заказа.

### ИИ‑консультант
- Чат‑бот или виджет (обычный и голосовой вариант — опционально) для помощи с подбором товаров, cross‑sell/up‑sell и ответов на частые вопросы; интеграция с API LLM (опционально настраиваемая база знаний).

### Производительность
- Быстрая загрузка страниц, кэширование, SEO‑оптимизация.

### Безопасность
- Защита данных клиентов, соответствие GDPR/локальным требованиям.

### Тестирование
- Unit и интеграционные тесты, CI/CD настроен (Vercel/GitHub Actions или аналог).

### Документация
- Развертывание, инструкции для админки, API документация.
